# FoundriOS Branding

Plaats hier alle branding assets en guidelines voor FoundriOS.

## Structuur

```
branding/
├── README.md           ← dit bestand
├── guidelines.md       ← kleurpalet, typografie, tone of voice
├── logo/               ← logo bestanden (SVG, PNG)
├── screenshots/        ← product screenshots voor marketing
└── assets/             ← overige visuele assets (iconen, illustraties)
```

## Gebruik

Deze map wordt gelezen door agents en skills die content, landing pages of marketingmateriaal genereren voor FoundriOS.
